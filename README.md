
Socially – Advanced Mobile Social Media Application

Socially is a feature-rich, cross-platform social media application built with Flutter. It features a sophisticated dual-user ecosystem (User & Admin) and utilizes a hybrid cloud architecture to handle real-time notifications, social interactions, and large-scale media storage.

🚀 Key Features
👥 Dual User Roles
User Level: Personal profiles, content creation, and social networking.

Admin Level: Administrative dashboard for content moderation and platform management.

📱 Social Engine
Post Management: Create, view, and share posts with support for high-quality media.

Interactions: Real-time Comments and Likes system.

Content Sharing: Seamless sharing functionality across the platform.

🔔 Real-time Notifications
Instant push notifications for social activities (likes, comments, and new followers).

Role-specific alerts for Admin actions and system updates.

☁️ Cloud Media & Data
Media Hosting: All images and videos are securely stored and served via Supabase Cloud Storage.

Hybrid Backend: Firebase handles secure Authentication, while Supabase manages the PostgreSQL relational database.

🛠️ Tech Stack
Frontend: Flutter (Dart)

Authentication: Firebase Auth

Database: Supabase (PostgreSQL)

Cloud Storage: Supabase Storage (for Videos & Images)

Push Notifications: Firebase Cloud Messaging (FCM) / Supabase Realtime
