## 🚀 Key Features

### 👥 Dual User Roles
* **User Level:** Personalized profiles, high-quality content creation, and social networking.
* **Admin Level:** Dedicated administrative dashboard for content moderation and platform health management.

### 📱 Social Engine
* **Post Management:** Create, view, and share posts with support for high-resolution images and videos.
* **Real-time Interactions:** Sophisticated Comments and Likes system powered by real-time streams.
* **Content Sharing:** Seamless cross-platform sharing functionality.

### 🔔 Real-time Notifications
* **Instant Push:** Automated alerts for social activities (likes, comments, and new followers) using FCM.
* **System Alerts:** Role-specific notifications for Admin actions and security updates.

---

## 🛠️ Tech Stack & Hybrid Cloud Architecture



* **Frontend:** Flutter (Dart) - Single codebase for Android & iOS.
* **Authentication:** Firebase Auth - Secure OAuth and email/password management.
* **Database:** Supabase (PostgreSQL) - Relational data management for complex social graphs.
* **Cloud Storage:** Supabase Storage - High-performance CDN for hosting videos and images.
* **Push Notifications:** Firebase Cloud Messaging (FCM) & Supabase Realtime.

---

## 🧩 Software Engineering Concepts Used

* **Hybrid Backend Design:** Leveraging the strengths of both Firebase (Auth/Notifications) and Supabase (SQL Database).
* **State Management:** Efficient UI updates using modern Flutter state management patterns.
* **Role-Based Access Control (RBAC):** Strict security rules separating User and Admin data access.
* **Lazy Loading:** Optimized media feeds to ensure low data consumption and high performance.
